BioSite Style Guide

This style guide provides an overview of the design principles, colors, typography, and UI for the Bio Site project to ensure consistency
across all visual and interactive elements.

Design Principles

Consistency

Ensure visual and functional consistency across all pages and components, facilitating a cohesive user experience.

Accessibility

Design for inclusivity, ensuring content is accessible to users with diverse abilities, including color contrast, keyboard navigation, and alt text.

Color

  • Colors Scheme: #FAEAB6, #999DBA, #B6BFFA, #A59B79, #464D7A, #3A4065
  • Background: #B6BFFA
  • Accent: #464D7A, #FAEAB6
  • Primary Text: #3A4065
  • Link/Highlight: #464D7A, #FAEAB6 (hover effect)

Typography

Font Family

  • Primary: Lora, serif
  • Usage: Body text, headings, buttons

Font Sizes

  • Headings:
    • h1: 2em
    • h2: 1.5em
    • h3: 1.17em
    • h4: 1em
  • Body Text: 1em (with line-height of 1.5 for readability)
  • Links and Buttons: 1em

Font Weight

  • Normal: 400
  • Bold: 700
  • Usage: Emphasize headings, important text, and actionable items.

UI Components

Buttons

  • Background Color: #464D7A
  • Padding: 10px 20px
  • Border Radius: 5px
  • Hover State: Background Color #FAEAB6, Text Color #464D7A

Links

  • Text Color: #464D7A
  • Hover State: Text decoration underline, Color #FAEAB6

Images

  • Border: 7px solid #FAEAB6
  • Hover State: Border Color #464D7A
  • Border Radius: 5% for soft edges

Navigation Bar

  • Background Color: #464D7A
  • Link Color: #B6BFFA
  • Hover State: Box-shadow beneath the text: 0 5px 0 0 #FAEAB6

Footer

  • Background Color: #464D7A
  • Text Color: #3A4065
  • Link Color: #FAEAB6
